# AI Dev Assistant

AI Dev Assistant is a multi-provider web app for analyzing code with AI.

Users can:
- explain code
- improve code
- find potential bugs

The app supports multiple providers through one interface:
- Ollama
- OpenAI-compatible APIs

## Features

- One UI for multiple AI providers
- Provider-specific settings
- Configurable model selection
- Explain / improve / bug analysis modes
- Backend provider abstraction for cleaner architecture

## Tech Stack

- Node.js
- Express
- Vanilla JavaScript
- HTML / CSS
- Ollama
- OpenAI-compatible APIs
